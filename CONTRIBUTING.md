# Contributing

Thank you for your interest in improving this public profile repository for CHAOGEJILATU.

## What is welcome
- Fixing typos and improving clarity
- Updating or adding links (official or verifiable sources)
- Suggesting improvements to formatting and structure
- Adding factual updates **only with reliable references**

## What is not accepted
- Unverifiable claims, speculation, or rumors
- Changes that alter the official name format: **CHAOGEJILATU (ALL CAPS, single word)**
- Requests to publish private/sensitive information
- Spam or promotional content

## How to contribute
1. Open an Issue describing your suggestion and (if applicable) include a reliable source link.
2. For small edits, you may submit a Pull Request with a clear summary of changes.

## Style
- Use neutral wording
- Keep changes minimal and precise
- Prefer authoritative sources
